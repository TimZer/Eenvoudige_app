Slaaptracker

Een eenvoudige PWA om je slaapuren bij te houden en statistieken te bekijken.

Functies
Slaapuren registreren (bedtijd & wektijd)
Statistieken (gemiddelde, min, max slaap)
Offline werken (PWA)
Data opslaan in de browser (LocalStorage)
Meertaligheid (NL / EN)

Projectstructuur
/opmaak
/javascript
/lang
index.html
manifest.json
service-worker.js

Talen
Nederlands (nl.json)
Engels (en.json)

Installeren
Android: “Toevoegen aan startscherm”
Desktop: Install-knop in browser
iPhone: “Zet op beginscherm”

Tech
HTML / CSS / JavaScript
Service Worker (offline support)
PWA (installable web app)

Let op
Vereist correcte icons (192x192 + 512x512)
Werkt alleen via HTTPS
